# Desafio de Projeto: Git/GitHub

## Descrição do Desafio
Este repositório foi criado para o desafio de projeto sobre **Git e GitHub**. O objetivo é demonstrar conhecimentos básicos de criação de repositório, commits e formatação de texto.

## Instruções do Desafio
1. Criar uma conta no GitHub.
2. Criar um repositório pessoal.
3. Adicionar este arquivo README.
4. Utilizar a sintaxe Markdown para estilizar o texto.

## O que aprendi
* Como criar repositórios públicos.
* Como usar o editor web (github.dev).
* Formatação básica de texto:
  * **Negrito**
  * *Itálico*
  * Listas
